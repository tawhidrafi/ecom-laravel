############ Laravel Ecommerce Website ############

A simple and modern eCommerce web application built with Laravel. This project includes user authentication, product management, cart functionality, and order processing with a clean UI.

# Features

User Authentication
Register
Login
Email Verification
Forgot / Reset Password
OTP Verification
Product Management
Product listing
Product details page
Categories
Shopping Experience
Add to cart
Update cart
Checkout flow
Orders
Place orders
Order history
Order status tracking
Admin Panel
Manage products
Manage orders
Manage users

# Tech Stack

Backend: Laravel
Frontend: Blade + Tailwind CSS
Database: MySQL
Authentication: Built from scratch
Version Control: Git
